Hello There!

  • Did you know: HTTP first came out in 1991?
    HTTP stands for Hypertext Transfer Protocol
    Before HTTP, people used smoke signals or IPoAC to communicate over long distances.

  • Did you know: HTTP/2 came out in 2015?
    With it came support for TLS which in turn enabled HTTPS, or Hypertext Transfer Protocol Secure

  • Did you know: HTTP/3 came out in 2022?
    HTTP/3 uses a different transport protocol called QUIC, while HTTP/1 and HTTP/2 use TCP.
